# C++ Practice

This repository contains my C++ practice while learning data structures and algorithms.

The goal is to improve problem-solving ability by solving many problems across different topics commonly used in competitive programming.

All programs in this repository are:

* Written in **C++**
* Developed using **Neovim**
* Compiled and tested on **Arch Linux**

The problems here mainly focus on:

* Basic algorithms
* Data structures
* Problem solving patterns used in competitive programming
* Preparing for platforms like Codeforces

This repository is part of my long-term effort to become a strong competitive programmer and build deeper intuition for algorithms and systems programming.
